Estudo de caso: VIOLAÇÃO DE DADOS
Mark trabalha para um fornecedor/agência de classificação de crédito e recentemente começou a trabalhar com um provedor de software de código aberto para criar aplicativos para a internet.

Os hackers exploraram uma vulnerabilidade no software de código aberto e os dados de aproximadamente 147 milhões de consumidores foram roubados.
Os hackers acessaram informações confidenciais como nomes, números de CPF, datas de nascimento, endereços e, em alguns casos, números de carteira de motorista, bem como detalhes de cartões de crédito.
Como resultado, a empresa de Mark está avaliando multas ou custos de compensação de mais de um bilhão de dólares.

Este exemplo reitera a importância de trabalhar com fornecedores confiáveis, que possam garantir a segurança e a integridade dos dados de uma empresa.
